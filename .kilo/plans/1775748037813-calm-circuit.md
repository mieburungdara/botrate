# Plan: Buat Branch Pure PHP (Hapus Semua Node.js)

## Tujuan
Membuat branch utama baru (`master-php-only`) yang hanya berisi kode Laravel PHP tanpa file Node.js sama sekali. Semua file dan direktori yang berhubungan dengan Node.js akan dihapus.

## Langkah-langkah yang akan dilakukan:

1.  **Buat branch baru** `master-php-only` dari master
2.  **Hapus semua file dan direktori Node.js:**
    - `NODE_SYSTEM/` (folder utama Node.js)
    - `package.json`
    - `package-lock.json`
    - `server.js`
    - `migrate.js`
    - `src/` (jika masih ada)
    - `webapp/` (jika masih ada)
    - `node_modules/` (jika masih ada)
3.  **Perbarui file dokumentasi:**
    - Edit `README.md` untuk menghapus referensi Node.js
    - Edit `DEPLOY_SHARED_HOSTING.md` untuk menghapus bagian Node.js
    - Hapus file dokumentasi yang khusus Node.js
4.  **Perbarui `.gitignore`** untuk menghapus entry Node.js
5.  **Commit semua perubahan**
6.  **Push branch ke GitHub**

## File yang akan dihapus:
- [ ] `NODE_SYSTEM/` directory
- [ ] `package.json`
- [ ] `package-lock.json`
- [ ] `server.js`
- [ ] `migrate.js`
- [ ] `TODO_SERVER.md`
- [ ] Semua referensi Node.js di dokumentasi

## Hasil akhir
Branch `master-php-only` akan menjadi branch pure Laravel PHP tanpa ketergantungan Node.js sama sekali, siap untuk deployment di shared hosting yang hanya mendukung PHP.
